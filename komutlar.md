## redis ##

```bash
# Start Redis
sudo systemctl start redis-server

# Check status
sudo systemctl status redis-server

# Enable on boot
sudo systemctl enable redis-server

# Test connection
redis-cli ping
```

## redis-cli ##

```bash
# Connect to Redis
redis-cli

# Set a key
SET mykey "Hello Redis"

# Get a key
GET mykey

# Exit
exit
```
