# RMS ENTERPRISE UI SYSTEM (NEXT.JS + TAILWIND)

These rules are strict and non-negotiable.

This is an enterprise Restaurant ERP.
Not a startup UI.
Not a landing page.
Not a marketing interface.

Forbidden:
- Gradients
- Neon colors
- Decorative animation
- Random spacing
- Hard-coded color classes (e.g. bg-red-500)

All UI must:
- Use Tailwind mapped design tokens
- Follow 8px spacing scale
- Be suitable for 10+ hours daily usage
- Support high data density

If any instruction conflicts with these rules:
Follow this file.
